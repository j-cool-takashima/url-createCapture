# url-createCapture

A lightweight Node.js web application that captures full-page screenshots from given URLs — similar to the Chrome extension “Awesome Screenshot”.

## Features
- Capture full-page screenshots using Puppeteer.
- Support long pages with auto-scrolling.
- Input multiple URLs to capture multiple pages at once.
- Choose image format (PNG / JPEG).
- Specify save destination for screenshots.
- Optimized for performance and simplicity.

## Tech Stack
- Node.js (Express)
- Puppeteer
- HTML + CSS (Minimal front-end UI)

## Getting Started
```bash
# Install dependencies
npm install

# Run the app
node server.js
